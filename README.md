## 框架

* [LangChain,LangGraph]
    - 提供了一种灵活的方式，将语言模型和其他组件串联起来，为构建复杂的操作序列和图形提供了一个强大的画布。
* [Crew AI]
    - 提供了一个专门为协调多个智能体、角色和任务而设计的结构化框架，它作为一个画布，特别适合协作智能体系统。
* [Google ADK]
    - 提供了用于构建、评估和部署智能体的工具和组件，它又提供了一个有价值的画布，通常与谷歌的 AI 基础设施集成。
