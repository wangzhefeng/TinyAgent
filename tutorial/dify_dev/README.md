# Dify 简介

* 工作流(workflow)
* 插件(plugin)
* 知识库(knowledge base)

Dify 内置了构建 LLM 应用所需的关键技术栈，包括：

* 对数百个模型的支持
* 直观的 Prompt 编排界面
* 高质量的 RAG 引擎
* 稳健的 Agent 框架
* 灵活的工作流
* 提供了一套易用的界面和 API
